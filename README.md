# Shopping-Cart

### E-commerce web application built with Laravel and Stripe API
an e-commerce website for adding items to a shopping cart, registred users can checkout the payment 
with a credit card and see a list of all the orders they bought.

# Usage
create a local database in mysql and name it shoppingcart, create the database tables
by executing the command: php artisan migrate
then seed the product table in the shoppingcart database
by executing the command: php artisan db:seed
